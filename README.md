altimu10-rpi
============

* This code is released under the GNU General Public License.
* Developed by Gilles Lasnier, based on a first implementation by Mark  Williams
* Created 25th March 2014

Description
===========

This program will enable your Raspberry Pi to get access to values from the 
Pololu AltIMU-10 intertial mesurement unit (IMU).

Hardware dependencies
=====================

- Raspberry Pi Model B
- Pololu AltIMU-10 

References
==========

Mark Williams's first implementation, available at:
https://github.com/mwilliams03/Raspberry-Gyro-Acc
